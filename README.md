# E-Payment System Backend

## Description
A robust electronic payment service inspired by platforms like Fawry, allowing users to pay for various services, apply discount coupons, and manage transactions with ease. This system includes user-friendly features such as secure JWT-based authentication, email verification, and a wallet for managing refunds and balances.

## Key Features

* JWT Authentication: Secure login with long-lived refresh tokens.
* Email Verification: Users activate their accounts via email before using the service.
* Password Security: User passwords are hashed using bcrypt with a custom salt generator.
* Transaction & Refunds: Users can manage transactions and request refunds, which are returned to their wallet.
* Wallet System: Each user has a personal wallet to manage funds.
* Admin Controls: Admins can create discount coupons, accept or reject refund requests, and manage user privileges.
* Authorization System: Prevents unauthorized access to admin-specific endpoints.
* Design Principles: The project follows SOLID principles and uses the Factory Design pattern for payment services.

## Tech Stack

* Languages & Frameworks: TypeScript, Node.js, NestJS
* Database: TypeORM, SQLite
* Authentication: Passport.js, JWT
* Validation & Transformation: class-transformer, class-validator
* Security: bcrypt for password hashing
* Email: Nodemailer for email verification

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
